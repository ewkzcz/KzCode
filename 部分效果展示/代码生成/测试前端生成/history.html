Agent 发展历程

从早期符号推理到现代大语言模型驱动的自主代理，Agent 技术经历了数十年演进。

1950s – 图灵测试与早期 AI

艾伦·图灵提出"机器能思考吗？"这一根本问题，并设计了图灵测试来评估机器智能。

1960s-70s – 符号主义与专家系统

基于符号逻辑的推理系统成为主流，诞生了 DENDRAL、MYCIN 等早期专家系统，奠定了知识表示与推理的基础。

1980s-90s – 分布式 AI 与多 Agent 系统

研究者开始关注多个智能体之间的协作与通信，提出了 BDI（信念-愿望-意图）模型，成为 Agent 理论的重要里程碑。

2000s-2010s – 强化学习与深度学习的崛起

深度 Q 网络（DQN）、AlphaGo 等突破展示了 Agent 在复杂环境中的学习能力，深度强化学习成为核心范式。

2020s 至今 – 大模型驱动的自主 Agent

ChatGPT、GPT-4、Claude 等大语言模型的出现，使 Agent 具备了自然语言理解、推理、规划与工具调用能力，LLM Agent 成为前沿热点。
